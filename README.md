# hea-ui

> hea-ui 是一款比较成熟的前端组件和解决方案，基于vue开发，后续会发展成vue react任选的方案 完美的解决了业务与视图之间的耦合关系组件化开发，效率更高，速度更快，无缝对接数据接口。


## 安装

#### NPM

```shell
npm install hea-ui --save  vant
```

## 快速上手

```js
import Vue from 'vue';
import HeaUi from 'hea-ui';
import 'hea-ui/lib/index.css';
Vue.use(Vant);
```
