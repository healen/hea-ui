# Ico 图表
----
### 基础用法
使用```name```属性来定义 Ioc 的样式。

<div class="demo-block">
	<div class="icolist">
	    <hea-ico name='goods'></hea-ico>
	    <hea-ico name='github' color="#ff0000"></hea-ico>
	    <hea-ico name='star'></hea-ico>
	</div>
</div>

### Attributes
| 参数      | 说明    | 类型      | 可选值       | 默认值   |
|----------|-------- |---------- |-------------  |-------- |
| name     | 图标名称   | string  | 已有图标库名称 |    —     |
| info     | 图标文字   | string    | -  |     —    |
| color    | 图标颜色   | string    | - |  -   |
